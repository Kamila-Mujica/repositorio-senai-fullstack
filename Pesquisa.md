# PESQUISA

-Pergunta 1 : **O que é programação?**
---
 Processo de criar instruções para que um computador execute tarefas. Essas instruções são escritas em uma linguagem de programação e servem para resolver problemas ou automatizar atividades.

 ---

-Pergunta 2 : **O que é um algoritmo**
---
 É uma sequência lógica e organizada de passos para resolver um problema ou realizar uma tarefa. Ele pode ser descrito em linguagem natural, fluxogramas ou código.

 ---

-Pergunta 3 : **Qual a diferença entre algoritmo e linguagem de programação?**
---
 Algoritmo: é a solução ou sequência de passos para resolver um problema. Já a
    Linguagem de programação: é a ferramenta usada para escrever o algoritmo de forma que o computador possa entendê-lo e executá-lo.

---

-Pregunta 4 : **O que é JavaScript?**
---
   É uma linguagem de programação amplamente utilizada para criar páginas web interativas, aplicações web, aplicativos móveis, servidores e muito mais.

---

-Pergunta 5 : **Onde o JavaScript pode ser utilizado atualmente?**
---
 O JavaScript pode ser utilizado em:

 Sites e aplicações web.
 Desenvolvimento de servidores.
 Aplicativos para celulares.
 Jogos.
 Aplicações desktop.
 Automação de tarefas.
 Desenvolvimento de APIs e sistemas em nuvem.

---
-Pergunta 6 : **O que é o Node.js?**
---
 É um ambiente de execução que permite rodar código JavaScript fora do navegador, diretamente no sistema operacional.

---

-Pergunta 7 : **Qual é a função do Node.js no desenvolvimento com JavaScript?**
---
 O Node.js permite usar JavaScript no lado do servidor, possibilitando criar APIs, servidores web, sistemas de back-end, automações e aplicações completas sem depender apenas do navegador.

-Pergunta 8 : **O que é o VS Code?**
---
 É um editor de código-fonte desenvolvido pela Microsoft. Ele oferece recursos como destaque de sintaxe, depuração, extensões e integração com sistemas de controle de versão.

 ---

-Pergunta 9 : **Qual a diferença entre um editor de texto comum e um editor de código?**
---
 Editor de texto comum: criado para editar textos simples (como bloco de notas).
 Editor de código: possui recursos específicos para programação, como:
        Colorização de sintaxe.
        Autocompletar código.
        Identificação de erros.
        Integração com ferramentas de desenvolvimento.

---
-Pergunta 10 : **O que é o terminal?**
---
 É uma interface baseada em comandos que permite interagir com o sistema operacional, executar programas, navegar por pastas e utilizar ferramentas de desenvolvimento.

---

-Pergunta 11 : **Para que serve o comando node arquivo.js?**
---
 Esse comando executa um arquivo JavaScript chamado arquivo.js usando o Node.js. O código contido no arquivo será interpretado e executado pelo ambiente Node.js.

---

-Pergunta 12 : **O que é uma variável?**
---
 Uma variável é um espaço nomeado na memória usado para armazenar dados que podem ser utilizados e manipulados durante a execução de um programa.

        Exemplo:

        let nome = "Maria";

---
-Pergunta 13 : **Qual a diferença entre let e const?**
---
-let: permite que o valor seja alterado posteriormente.
-const: cria uma constante, cujo valor não pode ser reatribuído após a criação.

---
-Pergunta 14 : **O que são tipos de dados? Cite exemplos.**
---
   Tipos de dados definem o tipo de informação que uma variável pode armazenar.

 -**Exemplos em JavaScript:**

 - **String (texto):**

     let nome = "João";

  -**Number (número):**

    let idade = 25;

 -**Boolean (verdadeiro ou falso):**

    let ativo = true;

---
-Pergunta 15 : **Qual a principal diferença entre JavaScript e TypeScript?**
---
 A principal diferença é que o TypeScript permite definir os tipos dos dados (texto, número, etc.), enquanto o JavaScript não exige isso.
