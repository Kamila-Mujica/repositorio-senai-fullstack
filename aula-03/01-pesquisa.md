# Perguntas
---
Pergunta 1 - **O que é uma estrutura condicional em programação? Dê um exemplo do cotidiano usando "se... então... senão".**
---
Uma estrutura condicional é usada para tomar decisões em um programa. Ela verifica uma condição e escolhe o que fazer.

Exemplo do cotidiano: Se estiver chovendo, então levo um guarda-chuva; senão, saio sem guarda-chuva.

---
Pergunta 2 - **Qual é a diferença entre usar apenas if e usar if/else?**
---
A diferença principal é que apenas o if verifica uma condição de forma independente, testando-a para decidir se executa um bloco de código ou não. O if/else cria uma relação de exclusão, ele executa uma ação se a condição for verdadeira, e define um caminho alternativo "caso contrário" se ela for falsa.

---
Pergunta 3 -  **Quando usamos else if? Crie um exemplo com três ou mais possibilidades**
---
Usamos o else if quando precisamos verificar três ou mais possibilidades mutuamente exclusivas em um fluxo de decisão. Ele permite testar uma nova condição apenas se a anterior for falsa. Se nenhuma for atendida, o else final é executado.


---
Pergunta 4 - **O que significa uma condição retornar true ou false?**
---
Significa simplesmente que ela foi testada e avaliada pelo sistema, resultando em um valor lógico binário. O true indica que o critério foi satisfeito, enquanto o false indica que o critério não foi alcançado.


---
Pergunta 5 - **Qual é a diferença entre == e === em JavaScript? Qual deles é mais recomendado?**
---
Em JavaScript, ambos servem para comparar valores, mas funcionam de forma diferente:

== (igualdade solta): compara os valores e pode converter os tipos automaticamente antes da comparação.

=== (igualdade estrita): compara os valores e os tipos, sem fazer conversões automáticas.


---
Perguntas 6 - **Para que servem os operadores &&, || e !? Explique com um exemplo para cada um.**
---
Eles permitem que o seu código tome decisões baseadas em múltiplos cenários.

- **Operador &&(E/END)**
Para que serve: Retorna true (verdadeiro) apenas se ambas as condições forem verdadeiras. Se pelo menos uma for falsa, o resultado será false.

Exemplo: Imagine que para entrar em uma festa, você precisa ter o ingresso e ser maior de idade.

---
- **Operador || (OU Lógico)**
Para que serve: Retorna true se pelo menos uma das condições for verdadeira. Ele só retorna false se todas as condições forem falsas.


---
- **Operador ! (Negação / NÃO Lógico)**
Para que serve: Serve para inverter o valor lógico de uma condição. Ele transforma true em false, e vice-versa.
